# Brandbase Engineering - Precision Plastic Injection Moulding

Engineering excellence in integrated manufacturing solutions.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

## Technologies

- **Frontend:** React, Vite, TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Animations:** Framer Motion
- **Icons:** Lucide React
